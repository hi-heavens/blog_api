The Blogging API

The general idea here is that the API has a general endpoint that shows a list of articles that have been created by different people, and anybody that calling this endpoint should be able to read a blog created by them or other users.

To see the sample requests, please access the Postman documentation HERE

  • /api/v1/user/signup - Send a POST request to register a user
  • /api/v1/user/login - Send a POST request to register a user
  • /api/v1/blogs - Returns all blog posts
  • /api/v1/blogs/create - Send a POST request to create a blog post
  • /api/v1/blogs/all - Returns all blog posts for a login user
  • /api/v1/blogs/:id - Returns a blog post with the provided ID
  • /api/v1/blogs/:id - When a PUT request is sent, a login user can update a blog post with the ID provided
  • /api/v1/blogs/:id - When a DELETE request is sent, a login user can delete a blog post with the ID provided

A token will be provided upon logging in successfully or successful user creation
